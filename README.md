# Análise de Complexidade de Algoritmos

**Professor:** Alexandre Montanha
**Disciplina:** Estruturas de Dados e Análise de Algoritmos

## Sobre a atividade

Este repositório contém a resolução de uma atividade cujo objetivo é analisar manualmente a complexidade de tempo de alguns algoritmos utilizando a notação Big-O.

A proposta da atividade foi observar a estrutura dos códigos fornecidos, identificar loops, chamadas recursivas e condicionais, e então determinar como o tempo de execução cresce conforme o tamanho da entrada aumenta.

Todas as análises foram feitas manualmente, sem o uso de ferramentas automáticas, com foco no raciocínio lógico sobre o comportamento dos algoritmos.

## Conteúdo do repositório

Os exercícios analisados incluem:

* Acesso direto a elementos de lista
* Percurso de listas
* Busca binária
* Comparação de pares de elementos
* Loops aninhados
* Crescimento logarítmico
* Recursão (Fibonacci)
* Algoritmos de ordenação
* Multiplicação de matrizes
* Algoritmo Merge Sort

Cada exercício contém:

* O código fornecido
* A análise da complexidade de tempo
* Uma breve justificativa explicando o raciocínio utilizado

## Objetivo

O objetivo principal desta atividade é desenvolver a capacidade de analisar algoritmos e compreender como diferentes estruturas de código impactam no desempenho conforme o tamanho da entrada aumenta.
